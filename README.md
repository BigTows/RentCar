## Курсовой проект на тему:
###  Создание и введение базы данных проката автомобилей


> Версия 0.4.1 Бета

## Возможности
- Заполнение таблиц
- Редактирование таблиц
- Удаление записей
- Аутентификаци
- Разграничение прав
- Просматривать точки на геокарте
- Заказывать машины
## Интсрументы
- Bootstrap 3
- Smarty
- MariaDB
- Apache 2
- Yandex Map
